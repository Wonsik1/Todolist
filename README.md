# 📚 Todolist

하루 할 일을 깔끔하게 정리하고 관리할 수 있는 투두리스트 앱입니다.

---

## 🧑‍💻 개발 환경

- iOS 17+
- Xcode 15.0+
- Swift 5.0
- UIKit + SnapKit
- Core Data

---

## 📌 주요 기능

### 🔍 할 일 추가/삭제하기

- 할 일 추가와 삭제를 할 수 있습니다.


### 🕘 할 일 목록 보기

- 할 일 목록들이 보여집니다.

### 📥 할 일 완료 체크/해제

 - 할 일을 완료했다면 체크하여 표시할 수 있습니다.
